# BUAA_jwangsy_blog
软院计网实验博客，由2321共创。发起者为rainel和Tengpaz。祝食用愉快~~

- 如果你找到了bug/有任何建议/想要加入共创，欢迎联系我：b0_224（微信号）
- 我们的博客采取 hackmd & github 双线更新的方式
  - hackmd 用于在实验过程中及时修改 bug & 人多力量大。大家遇到bug可以随手即使修改，互通有无，互相帮助。因此，它会是更新**最及时**的。[查看hackmd团队首页](https://hackmd.io/@jwsy)，你可以在里面看到所有帖子，登录之后即可参与共创。
  - github 用于长期保存/和软院仓库连接，现已更新完全部内容。

关于hackmd的一些tips：
1. 似乎需要挂梯子访问
2. 共同编辑需要登录账号，否则似乎只能查看
3. 遇到错误的代码/步骤欢迎直接进行修改，如果有不确定的也可以采用 hackmd 自带的评论（非常好使，甚至可以选中文字来进行评论，但是似乎无法选中代码进行评论）
4. 如果有更好用的markdown共享网站/更多的建议欢迎在评论区留言
